# Calculadora de Ciclo Menstrual e Período Fértil

Aplicativo educativo em Python que estima o dia da próxima menstruação, o período fértil e possíveis atrasos, com base na data da última menstruação informada pela usuária.

---

## Como funciona
O programa solicita:
- Data da **última menstruação** (formato DD/MM/AAAA)
- Tipo de ciclo:
  - **N** → ciclo normal de 28 dias
  - **E** → ciclo específico (20 a 40 dias)

Após isso, calcula:
- Próxima menstruação estimada  
- Intervalo do período fértil  
- Datas-limite para atraso leve e grave  
- Mensagem educativa conforme o estado atual

---
